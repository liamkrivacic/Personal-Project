# Personal Portfolio

Next.js portfolio site for Liam Krivacic, built around a plasma/orbital project map.

## Run Locally

```powershell
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Useful Commands

```powershell
npm test
npm run lint
npm run build
```

## Current Direction

The homepage uses a dark RF/plasma-inspired field with compact project capsules orbiting
a central gravity well. Capsules can be selected, dragged, thrown, and returned to their
stable orbit with a spring/damping motion model.
